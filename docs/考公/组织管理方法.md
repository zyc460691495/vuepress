## 回应类
### 答题逻辑
1. 开头
   1. 问好
   2. 表明身份
   3. 感谢
   4. 道歉
   5. 承诺
2. 承接
3. 解释
   1. 肯定当前工作
   2. 解释原因（客观+主观）
4. 解决
   1. 问题一
   2. 问题二
5. 结尾
   - 询问式：还有哪些疑问
   - 承诺式、保证式：我们向你保证、承诺，您提出的问题后续给予满意答复，在以后的工作中
   - 呼吁式：我也想对大家发起号召
   - 感谢式：感谢提出的宝贵意见，提出了更高要求

### EXP01
>某地正在进行一场电视问政直播，水质检测部门的负责人说护城河的水污染防治效果好，取得了突破性成绩，现在已经没有污染了。
> 此时，现场突然有位群众拿出一瓶护城河的税，还说到：“你说治理好了，那请你把这瓶水喝下去”，现场一片哗然。
> 
> 面对这种情况，如果你是该负责人，会怎么样回应？请现场模拟

**答案来源于材料中，材料中有什么？**

对象：直播间观众、群众、现场群众

工作：水质检测，水污染防治好在哪？取得了什么突破性成绩？

身份：部门负责人

质疑：治理好了？喝水？





1. 开头
   1. 问好：这位同志你好，  
   2. 表明身份：我是咱们水质检测部门的负责人，
   3. 感谢：首先感谢你提出的宝贵意见，
   4. 道歉：我们工作中确实有一些做的不到位的地方，
   5. 承诺：在这里我向你承诺今天提出的疑问，我一定给予你满意的答复。
2. 承接：关于水污染治理的问题，下面我向你具体解释一下
3. 解释
   1. 肯定：我们当前的水污染治理工作目前取得了一定的成果，一方面河流变得更清澈、河道垃圾污染物更少了、异味也消失了；
      另一方面，河流的水质也更好了，鱼也变多了，沿岸花草树木也更茂盛。
   2. 至于您刚刚提到水能不能喝的问题，我这里给出的答复是不能喝，这里我要好好给你解释一下，
      我们平时的饮用水和治理好的河水的标准不一样，河水中含有各种细菌微生物、杂志，喝下去会影响我们的身体健康。
4. 对策：水是生命之源，水污染问题相信大家看在眼里痛在心里。请各位市民朋友们放心，我们也制定了一些解决措施
   1. 我们加派人员不定期巡查河流治理情况；
   2. 我们通过安装监控、派出无人机巡查的方式，扩大监测范围；
   3. 我们畅通了群众监督举报渠道，大家可以登录我们部门的网站进行举报投诉；
5. 结尾：在这里，我也想对现场的群众和直播间的观众发起呼吁，保护环境是我们每个人应尽的义务，
   希望大家在水污染防治中贡献自己的一份力量，共同建设美丽家园。



### EXP02

>一天，市民老张到主管部门反映情况，对接待他的工作人员小刘说：“我在范家桥观景时，经常看到有人把果皮和纸巾等杂物扔到河里
> 。今天我在劝说一位市民时，他还损了我一顿。你们为什么不在桥上装一个透明防护网。”

**答案来源于材料中，材料中有什么？**

对象：老张，热心群众

诉求：经常有人扔垃圾？为什么不装防护网？

身份：小刘、主管部门接待人员

其他：劝说被损

1. 开头
   1. 问好：张大爷你好，
   2. 表明身份：我是接待您的工作人员小刘，您先消消气，别激动
   3. 感谢：感谢您平时劝说市民的行为和提出的宝贵意见
   4. 道歉：我们在目前的工作中确实还存在一些做的不到位的地方，这里先跟您道个歉
   5. 我们承诺您今天反映的问题我们会及时跟进处理，
2. 承接：下面就您反映的问题，我给你解释一下
3. 解释
   1. 肯定：您的做法是值得肯定的， 一方面保护环境是每个人的责任，您不仅能规范自己的行为还能劝说他人，我们得向您学习。
      另一方面，您能来我们单位反映问题，并提出了宝贵意见，我们非常感谢
   2. 至于您提到的经常有人乱扔垃圾到河里的问题，我们一定会派出专人调查，及时开展整治。然后是您提出的装防护网的建议，
      这个因为牵扯到多个部分，需要我们召开会议讨论一下，也会将结果及时通知您。
4. 爱护环境是我们每个人的义务，为了减少乱扔垃圾的行为，我们也会采取一些措施的
   1. 一方面，我们会在桥上设置警示标语，增设垃圾桶，提醒人们不要随手乱扔垃圾
   2. 另一方面，我们也会畅通监督举报渠道，您可以打电话或者在我们网站上举报投诉不文明行为，我们也会对相应人员进行处罚
5. 结尾：张大爷，在这里我向您表示衷心的感谢，您提出的相应问题，我们一定会尽快处理并通知结果，希望您能满意。







## 劝说
### 答题逻辑

1. 开头
   1. 问好
   2. 表明身份
   3. 表明目的
2. 理解、肯定、换位思考
3. 说理
   1. 不做有什么好处
   2. 做了有什么坏处
4. 对策
5. 警示（警告）
6. 结尾


### EXP01

>现在无人机广泛流行、用的人非常多，摄影爱好者经常用于摄影航拍，但是对于无人机并没有相关规定，无人机会给人民带来生命财产安全。
> 黑飞现象比较严重。
> 
> 如果你是景区工作人员，看到有人航拍，你会怎么劝说？请现场模拟

**答案来源于材料中，材料中有什么？**

对象：摄影爱好者

用途：航拍摄影

问题：没有相关规定

危害：生命财产安全

身份：景区工作人员

1. 开头
   1. 问好：同志你好，
   2. 表明身份：我是咱们景区的工作人员
   3. 这边看到您在使用无人机航拍，麻烦您赶快停下，具体原因我稍后给您解释一下
2. 理解：我非常能理解您的心情，您来我们景区使用无人机航拍摄影，一方面是对我们景区的认可，另一方面也是为了拍出更多好看的照片，
   还能帮助宣传我们景区。
3. 说理：但是我们还是要跟您解释一下为什么不能进行无人机飞行
   1. 一方面，在景区内有很多动物，贸然使用无人机飞行会干扰他们的正常生活；
   2. 另一方面，而且景区内还有许多游客，如果不幸发生无人机坠落不仅会危害他人生命健康，也会对您造成财产损失。
4. 这里对您造成的不便还请谅解，但是考虑到广大摄影爱好者，我们也做了一些改进
   1. 一方面，我们正在出台相关无人机飞行规定，在景区内规划出可飞区域，满足无人机航拍需要；
   2. 另一方面，我们也为摄影爱好者免费提供相应景区的视频和图片，也可以向您推荐一些可以进行无人机飞行的周边景区。
5. 所以，为了营造良好的景区运营环境，还请您多多理解配合，
   如果您还飞行的话，我们景区会采取措施干扰无人机，也会追究您的相应责任，这都是我们不希望看到的局面。
6. 美丽的景区是大家共同努力的结果，如果您后续还有任何疑问，这是我们的联系方式，随时都可以联系我们，我们一定会及时给予您准确答复。


### EXP02

> 村民老王将一块村集体废弃池塘化为己用，开发成鱼塘，每年收益三万余元。现在政府规划决定将鱼塘收回，用作建设村民活动广场，
> 老王不同意，声称如果改就要经济补偿。
> 
> 你作为镇里驻村工作人员，你将怎么劝说老王，请现场模拟

**答案来源于材料中，材料中有什么？**

对象：村民老王

问题：集体化为己用

用途：回收建设广场

危害：生命财产安全

身份：驻村工作人员

1. 开头
   1. 问好：王大爷您好
   2. 表明身份：我是镇里的驻村工作人员
   3. 目的：今天来拜访您是想和你商量一下收回村集体鱼塘建设活动广场的事情
2. 理解
   我非常能理解您的想法，之前村集体池塘废弃了，您将其开发成鱼塘也废了不少功夫，肯定也投入一些成本，
   现在改收回的话，您每年鱼塘收益也没了，索要经济补偿也是人之常情。
3. 说理：但是，我还是要跟您解释一下为什么要回收鱼塘
   1. 一方面，咱们村缺少村民娱乐区域，现在把鱼塘改成活动广场，大伙茶余饭后有个活动的地方，对村里来说是一件好事
   2. 另一方面，鱼塘本来就是村里集体的，这些年您靠着鱼塘每年也有不少收益了，
      按理说这都是集体收益，您一直占为己有的话，别的村民也有意见，还会影响邻里关系，您说，是不是
4. 对策：所以，王大爷还请您多多理解，为了避免以后的此类问题，我们也提出了一些改进措施：
   1. 对村集体资产建立制度化管理，未经允许不得私自占用经营
   2. 另外，村西边还有一块空地，地方更大，我们打算把它开发成鱼塘出租给您经营
5. 警示：王大爷还是希望您多多支持我们工作，毕竟建设活动广场是村民大家伙的福利，如果您还是不同意的话，我们还会追究您私自占用经营的责任，
   这是我们都不希望看到的局面。
6. 总结：王大爷，还是希望您多理解，您还有任何疑虑都可以告诉我，我一定给您妥善解决
